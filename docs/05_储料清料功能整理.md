# 注塑机储料/清料功能

## 1. 概述

### 1.1 功能简介

<div class="info-card">
  <div class="icon">🔄</div>
  <strong>核心功能</strong>
  <p>储料功能是注塑机的重要功能之一，负责将塑料颗粒通过料筒加热熔融并输送到注射区域，为后续的射出过程做准备。储料过程的控制精度直接影响产品的重量一致性和成型质量。</p>
</div>

### 1.2 工艺特点

<div class="feature-block">
  <h4>工艺特性</h4>
  <ul>
    <li><strong>多段储料控制</strong>：支持8段储料工艺参数，实现精确的物料计量</li>
    <li><strong>背压控制</strong>：支持储料背压控制，确保物料均匀熔融</li>
    <li><strong>位置控制</strong>：支持电子尺和行程开关两种位置控制方式</li>
    <li><strong>斜率控制</strong>：支持压力和速度的启动/停止斜率控制</li>
    <li><strong>安全机制</strong>：包含超时保护、状态互锁等多重安全保障</li>
    <li><strong>平台兼容性</strong>：支持Luban平台（基于Beremiz二次开发）运行，采用标准IEC 61131-3 ST语法实现</li>
  </ul>
</div>

### 1.3 技术架构

本功能采用分层架构设计，参考研发部提供的液压系统建模方案，结合倍福TF8560塑料技术功能标准，实现模块化、标准化设计。

## 2. 核心控制机制

### 2.1 储料状态机 (E_MeterState)

<div class="info-card">
  <div class="icon">⚡</div>
  <strong>状态机控制</strong>
  <p>储料功能采用统一状态机管理，包含空闲、初始化、储料中和储料完成等状态。</p>
</div>

```mermaid
flowchart TD
    Start[初始状态 eState_Idle] -->|启动信号| Init[初始化状态 eState_Init]
    Init -->|初始化完成| Idle[空闲状态 eState_Idle]
    Idle -->|储料模式启动| Metering[储料中 eState_Metering]
    Metering -->|储料8段完成| Metered[储料完成 eState_Metered]
    Metered -->|复位| Idle
    Metering -->|错误发生| Error[错误状态 eState_Error]
    Error -->|复位命令| Idle

    class Start,Init,Idle,Metering,Metered state
    class Error error
```

#### 状态说明

| 状态值 | 名称            | 说明        |
| ------ | --------------- | ----------- |
| 0      | eState_Idle     | 空闲状态    |
| 1      | eState_Init     | 初始化状态  |
| 2      | eState_Metering | 储料中(8段) |
| 3      | eState_Metered  | 储料完成    |
| 4      | eState_Error    | 错误状态    |

### 2.2 控制命令机制

| 命令   | 说明     | 响应             |
| ------ | -------- | ---------------- |
| bStart | 启动命令 | 启动储料动作     |
| bStop  | 停止命令 | 有减速停止       |
| bEStop | 急停命令 | 立即停止，无减速 |
| bReset | 复位命令 | 重置错误状态     |

### 2.3 储料方式选择 (uiMeterMode)

| 方式值 | 名称       | 说明                   |
| ------ | ---------- | ---------------------- |
| 0      | 电子尺模式 | 通过电子尺位置反馈控制 |
| 1      | 行程模式   | 通过行程开关控制       |

## 4. 控制流程

### 4.1 储料过程流程

#### 4.1.1 储料流程示意图

```mermaid
flowchart TD
    Start[初始状态] -->|启动信号| Init[初始化]
    Init -->|初始化完成| Idle[空闲状态]
    Idle -->|储料模式启动| MeterMode[储料等级选择 1-N级]

    subgraph 储料 N级 循环
        direction TB
        MeterMode --> InitMeter[初始化当前等级=1]
        InitMeter --> ExecuteMeter[执行储料等级 当前等级]
        ExecuteMeter -->|位置到达或时间到| CheckMeter{当前等级 ≤ 选择等级?}
        CheckMeter -->|否| EndMeterLoop[结束储料循环]
        CheckMeter -->|是| IncrementMeter[当前等级+1]
        IncrementMeter --> ExecuteMeter
    end

    EndMeterLoop -->|储料完成| Metered[储料完成]
    Metered -->|复位| Idle

    MeterMode -->|错误| Error[错误状态]
    ExecuteMeter -->|超时| Error

    Error -->|复位| Idle

    class InitMeter,ExecuteMeter,IncrementMeter,EndMeterLoop step
    class MeterMode,Start,Init,Idle,Metered complete
    class CheckMeter decision
    class Error error
```

#### 4.1.2 储料流程序列图

```mermaid
sequenceDiagram
    participant 系统
    participant 储料控制
    participant 执行机构

    系统->>储料控制: 启动信号 (bStart=TRUE)
    储料控制->>储料控制: 初始化 (eState_Init)
    储料控制->>储料控制: 储料等级选择 (1-N级)
    储料控制->>储料控制: 初始化当前等级=1

    loop 储料等级循环
        储料控制->>执行机构: 执行储料等级(当前等级)
        alt 位置到达或时间到
            执行机构-->>储料控制: 位置到达或时间到信号
        else 超时
            执行机构-->>储料控制: 超时信号
            储料控制->>储料控制: 错误处理 (eState_Error)
        end
        储料控制->>储料控制: 当前等级+1
        储料控制->>储料控制: 检查是否达到选择等级
    end

    储料控制-->>系统: 完成信号 (bDone=TRUE, bMetered=TRUE)
```

> ⚠️ **重要说明**：
>
> 1. 储料等级段数可通过 `uiMeterSegCnt`参数设定（1-8段）
> 2. 储料过程支持背压控制，可通过 `uiBackPres`参数设置

## 8. 安全保护机制

### 8.1 超时保护

- **储料限制时间保护**：监控整个储料过程，超过设定总时间则报警
- **各段超时保护**：各段工艺参数都应有合理的时间限制

### 8.2 位置保护

- **电子尺位置监控**：实时监控螺杆位置
- **位置极限保护**：防止超出机械行程极限

### 8.3 状态互锁

- **状态机互锁**：各状态间应有明确的转换条件，防止异常跳转
- **命令互锁**：启动、停止、急停命令应有优先级处理

## 10. 参数调整指南

### 10.1 储料参数调整

<div class="info-card">
  <div class="icon">💡</div>
  <strong>调整建议</strong>
  <p>储料参数的调整应根据产品材质、模具结构和制品要求进行优化。</p>
</div>

1. **背压调整**：根据物料特性和塑化要求选择合适的背压

   - 背压有助于物料均匀熔融
   - 背压过大可能导致温度升高和降解
2. **速度调整**：根据填充阶段调整速度

   - 初始阶段速度较慢，保证物料平稳进入料筒
   - 中段速度加快，提高生产效率
3. **位置调整**：电子尺位置用于精确控制各段切换点

   - 确保各段位置设置合理，保证计量精度

### 10.2 背压控制调整

1. **背压作用**：在熔融过程中建立背压，帮助物料均匀混合和熔融
2. **背压设置**：根据材料特性调整，过高会导致温度升高
3. **零背压选项**：某些材料可能需要零背压控制

## 11. 调试与故障排除

### 11.1 常见问题与解决方案

| 问题现象     | 可能原因               | 解决建议                 |
| ------------ | ---------------------- | ------------------------ |
| 储料不足     | 背压参数设置过低       | 检查并调整背压参数       |
| 制品重量不稳 | 储料位置不一致         | 检查电子尺信号和位置参数 |
| 储料超时     | 各段时间设置过长       | 优化各段时间参数         |
| 物料降解     | 背压过高或储料时间过长 | 降低背压和储料时间       |

### 11.2 报警处理

| 报警代码 | 说明     | 处理方法                                 |
| -------- | -------- | ---------------------------------------- |
| 1001     | 储料超时 | 检查储料各段时间设置，检查螺杆和料筒状态 |
| 1002     | 位置超限 | 检查储料位置参数是否超出机械行程         |

| 参数名称 | 地址  | 说明             |
| -------- | ----- | ---------------- |
| 清料射出 | D0360 | 清料射出压力     |
| 清料射退 | D0350 | 清料射退压力     |
| 自动清料 | D0393 | 自动清料功能开关 |
| 清料次数 | D0394 | 自动清料次数     |

### 13.3 清料流程

```mermaid
flowchart TD
    Start[开始清料] --> CheckMode{自动清料?}
    CheckMode -->|是| AutoMode[执行N次自动清料]
    CheckMode -->|否| ManualMode[执行手动清料]
    AutoMode --> PurgeInject[清料射出]
    ManualMode --> PurgeInject
    PurgeInject --> PurgeRetract[清料射退]
    PurgeRetract --> CheckCount{清料次数 < N?}
    CheckCount -->|是| PurgeInject
    CheckCount -->|否| Complete[清料完成]
```

### 13.4 阀门输出

| 阶段     | 射退阀 | 储料阀 | 说明                 |
| -------- | ------ | ------ | -------------------- |
| 清料射出 | FALSE  | FALSE  | 射出阀打开，物料射出 |
| 清料射退 | TRUE   | FALSE  | 射退阀打开，防止漏料 |
