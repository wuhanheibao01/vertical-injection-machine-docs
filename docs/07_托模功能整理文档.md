# 注塑机托模功能整理文档

## 1. 功能概述
托模功能是注塑机的重要组成部分，主要负责控制模具顶针的前进和后退动作，实现制品的脱模。托模动作的精确控制对于制品的质量和生产效率具有重要影响，需要根据制品特性和模具要求进行参数调整。

### 3.2 托模参数设定流程
1. 在触摸屏主界面，按[座台/托模]键进入座台/托模设定页面
2. 选择托模参数区域
3. 解锁资料锁以允许参数修改
4. 使用光标键导航到需要修改的参数项
5. 输入新的参数值
6. 按[确认]键保存参数设置
7. 锁定资料锁以防止误操作

### 3.3 手动操作流程
1. 将机器切换到手动模式
2. 在主界面或功能键区，按[托进]键，托模将前进
3. 按[托退]键，托模将后退
4. 可通过[手动/自动]键切换操作模式

### 3.4 自动操作流程
1. 将机器切换到半自动或全自动模式
2. 确保托模参数已正确设定
3. 当满足自动启动条件时（通常是开模完成后），托模将自动启动
4. 托模将按照设定的次数和参数完成托模循环
5. 循环完成后，托模将停止在后退终点位置

### 3.5 低压保护操作说明
1. 确保低压保护参数已正确设定
2. 托模前进时，当到达低压保护位置时，系统自动切换到低压保护参数
3. 如果在低压保护时间内未到达前进终点，系统将报警并停止动作
4. 发生低压保护异常时，检查制品是否卡住，模具是否正确安装

## 4. 控制逻辑

### 4.1 托模动作状态流程图
开始 → 检查托模当前位置 → 根据命令执行相应动作 → 监控动作完成 → 结束

### 4.2 托模动作详细流程
1. 托模前进流程：
   - 接收托模前进命令
   - 检查托模是否在后退终点位置
   - 启动托模快速前进（使用托进压力和流量参数）
   - 到达低压保护位置时，切换到低压保护参数
   - 到达前进终点位置或超时后，停止动作
   - 进入托进延时阶段

2. 托模后退流程：
   - 接收托模后退命令或托进延时结束
   - 检查托模是否在前进终点位置
   - 启动托模后退（使用托退压力和流量参数）
   - 到达后退终点位置或超时后，停止动作
   - 进入托退延时阶段

3. 托模循环流程：
   - 接收启动托模循环命令
   - 执行托模前进流程
   - 执行托模后退流程
   - 更新托模计数
   - 检查是否完成设定次数
   - 如果未完成，继续下一次循环；如果完成，循环结束

## 5. 参数调整建议

### 5.1 压力参数调整原则
- **托进压力**：应根据制品大小和脱模难度设置，确保能够顺利顶出制品，但避免压力过高导致制品变形或模具损坏
- **托退压力**：通常设置略高于托进压力，确保顶针可靠退回
- **低压保护压力**：应设置较低，用于检测障碍物或制品卡住情况

### 5.2 流量参数调整原则
- **托进流量**：影响顶出速度，应根据制品特性调整，避免速度过快导致制品损坏
- **托退流量**：通常设置较高，以提高生产效率
- **流量与压力匹配**：流量参数应与压力参数匹配，确保平稳动作

### 5.3 位置和时间参数调整原则
- **托进位置**：应根据模具顶出行程设置，确保制品完全顶出
- **托退位置**：应设置为零位，确保顶针完全退回模具内
- **托模时间**：应设置为正常操作时间的1.5-2倍，用于检测异常情况
- **延时时间**：托进延时应足够长以确保制品掉落，托退延时可较短

### 5.4 托模次数调整原则
- **单模制品**：通常设置为1次
- **复杂制品**：可能需要多次顶出以避免制品卡在模具中
- **特殊工艺**：根据实际需要调整托模次数

#### 6.2.2 功能块方法实现

```st
// FB_Ejector方法实现
METHOD Execute : BOOL
    // 初始化
    Error := FALSE;
    ErrorCode := 0;
    CommandComplete := FALSE;
    
    // 检测启动命令上升沿
    StartCmdEdge := (NOT PrevStartCmd) AND StartCmd;
    PrevStartCmd := StartCmd;
    
    // 如果功能未使能，停止所有动作
    IF NOT FunctionEnable THEN
        StopMotion();
        RETURN FALSE;
    END IF;
    
    // 自动模式处理
    IF AutoMode THEN
        RETURN HandleAutoMode();
    // 手动模式处理
    ELSIF ManualMode THEN
        RETURN HandleManualMode();
    END IF;
    
    RETURN FALSE;
END_METHOD

METHOD HandleAutoMode : BOOL
    // 启动托模循环
    IF StartCmdEdge THEN
        StartEjectorCycle();
    END IF;
    
    // 处理托进阶段
    IF AdvStage THEN
        // 检查是否到达低压位置
        IF LowStage THEN
            // 低压阶段
            PressureOut := LowPressure;
            FlowOut := LowFlow;
            
            // 低压阶段计时
            LowTimer(IN := TRUE, PT := T#100MS * TO_REAL(LowTimeLimit));
            
            // 检查是否到达托进终点
            IF AtAdvEnd THEN
                // 切换到托进延时阶段
                AdvStage := FALSE;
                LowStage := FALSE;
                AdvDelayStage := TRUE;
                AdvTimer(IN := FALSE);
                LowTimer(IN := FALSE);
                
                // 启动托进延时计时器
                AdvDelayTimer(IN := TRUE, PT := T#100MS * TO_REAL(AdvDelayTime));
            ELSIF LowTimer.Q THEN
                // 低压超时错误
                Error := TRUE;
                ErrorCode := 101; // 低压超时
                StopMotion();
                RETURN FALSE;
            END IF;
        ELSE
            // 快速托进阶段
            PressureOut := AdvPressure;
            FlowOut := AdvFlow;
            
            // 托进计时
            AdvTimer(IN := TRUE, PT := T#100MS * TO_REAL(AdvTimeLimit));
            
            // 检查是否到达低压位置
            IF AtLowPos THEN
                LowStage := TRUE;
            // 检查是否到达托进终点
            ELSIF AtAdvEnd THEN
                // 切换到托进延时阶段
                AdvStage := FALSE;
                AdvDelayStage := TRUE;
                AdvTimer(IN := FALSE);
                
                // 启动托进延时计时器
                AdvDelayTimer(IN := TRUE, PT := T#100MS * TO_REAL(AdvDelayTime));
            ELSIF AdvTimer.Q THEN
                // 托进超时错误
                Error := TRUE;
                ErrorCode := 100; // 托进超时
                StopMotion();
                RETURN FALSE;
            END IF;
        END IF;
    // 处理托进延时阶段
    ELSIF AdvDelayStage THEN
        IF AdvDelayTimer.Q THEN
            // 托进延时结束，切换到托退阶段
            AdvDelayStage := FALSE;
            RetStage := TRUE;
            AdvDelayTimer(IN := FALSE);
        END IF;
    // 处理托退阶段
    ELSIF RetStage THEN
        PressureOut := RetPressure;
        FlowOut := RetFlow;
        
        // 托退计时
        RetTimer(IN := TRUE, PT := T#100MS * TO_REAL(RetTimeLimit));
        
        // 检查是否到达托退终点
        IF AtRetEnd THEN
            // 切换到托退延时阶段
            RetStage := FALSE;
            RetDelayStage := TRUE;
            RetTimer(IN := FALSE);
            
            // 启动托退延时计时器
            RetDelayTimer(IN := TRUE, PT := T#100MS * TO_REAL(RetDelayTime));
        ELSIF RetTimer.Q THEN
            // 托退超时错误
            Error := TRUE;
            ErrorCode := 102; // 托退超时
            StopMotion();
            RETURN FALSE;
        END IF;
    // 处理托退延时阶段
    ELSIF RetDelayStage THEN
        IF RetDelayTimer.Q THEN
            // 托退延时结束
            RetDelayStage := FALSE;
            CycleComplete := TRUE;
            
            // 更新计数器
            CycleCounter := CycleCounter + 1;
            CurrentCount := CycleCounter;
            
            // 检查是否完成所有循环
            IF CycleCounter >= CycleCount THEN
                // 完成所有循环
                CycleComplete := TRUE;
                CommandComplete := TRUE;
                StopMotion();
                RETURN TRUE;
            ELSE
                // 继续下一次循环
                AdvStage := TRUE;
                LowStage := FALSE;
            END IF;
        END IF;
    END IF;
    
    RETURN TRUE;
END_METHOD

METHOD HandleManualMode : BOOL
    // 手动托进命令
    IF AdvCmd AND NOT AtAdvEnd THEN
        AdvOut := TRUE;
        RetOut := FALSE;
        PressureOut := AdvPressure;
        FlowOut := AdvFlow;
        AdvanceInProgress := TRUE;
        RetreatInProgress := FALSE;
    // 手动托退命令
    ELSIF RetCmd AND NOT AtRetEnd THEN
        AdvOut := FALSE;
        RetOut := TRUE;
        PressureOut := RetPressure;
        FlowOut := RetFlow;
        AdvanceInProgress := FALSE;
        RetreatInProgress := TRUE;
    // 无命令时停止
    ELSE
        IF AdvanceInProgress OR RetreatInProgress THEN
            StopMotion();
        END IF;
    END IF;
    
    RETURN TRUE;
END_METHOD

METHOD StartEjectorCycle : BOOL
    // 停止当前动作
    StopMotion();
    
    // 重置计数器
    CycleCounter := 0;
    CurrentCount := 0;
    
    // 启动托进阶段
    AdvOut := TRUE;
    RetOut := FALSE;
    PressureOut := AdvPressure;
    FlowOut := AdvFlow;
    
    // 更新状态
    AdvStage := TRUE;
    LowStage := FALSE;
    InProgress := TRUE;
    
    // 重置计时器
    AdvTimer(IN := FALSE);
    RetTimer(IN := FALSE);

## 7. MODBUS通信适配说明

### 7.1 参数类型适配

为了与MODBUS通信协议保持一致，所有托模参数均采用INT类型：

- **压力参数**：PLC内部存储为bar×10的整数，如350表示35.0bar
- **流量参数**：PLC内部存储为%×10的整数，如300表示30.0%
- **位置参数**：PLC内部存储为mm×10的整数，如800表示80.0mm
- **时间参数**：PLC内部存储为s×10的整数，如25表示2.5s

### 7.2 数据转换说明

1. **HMI到PLC**：HMI显示的浮点数值（如35.0bar）乘以10后以整数形式（如350）通过MODBUS写入PLC
2. **PLC到HMI**：PLC内部的整数值（如350）通过MODBUS读取后除以10，在HMI上显示为浮点数值（如35.0bar）
3. **PLC内部计算**：在需要使用实数进行计算的地方（如计时器PT赋值），使用TO_REAL函数将INT类型转换为REAL类型

### 7.3 Timer参数处理示例

// 正确的Timer参数处理方式（使用TO_REAL转换）
AdvTimer(IN := TRUE, PT := T#100MS * TO_REAL(AdvTimeLimit));

## 8. 功能块实现要求

1. **功能完整性**：FB_Ejector功能块应实现所有托模相关的控制逻辑，包括前进、后退、延时、计数和保护等功能
2. **错误处理**：功能块应包含完善的错误检测和处理机制，如超时检测、位置检测等
3. **参数一致性**：确保所有参数类型与MODBUS通信协议保持一致，统一使用INT类型
4. **数据转换**：在需要进行计算的地方，必须使用TO_REAL函数将INT类型转换为REAL类型
5. **安全机制**：实现低压保护功能，防止制品卡住或模具损坏

## 9. 注意事项

1. **参数一致性**：触摸屏上修改的参数应实时同步到PLC程序中，确保参数一致性
2. **参数存储**：重要参数应保存在PLC的掉电保持区域，确保断电后参数不丢失
3. **参数限制**：应对关键参数设置合理的上下限，防止输入错误导致设备损坏
4. **位置精度**：托模位置调整应准确，避免顶针不到位或过度顶出
5. **低压保护**：托模低压保护是防止制品卡住或模具损坏的重要机制，应正确设置
6. **托模与开模协调**：托模动作应在开模完成后执行，避免干涉
7. **计数功能**：应确保托模计数准确，避免多顶或少顶
8. **数据转换**：在需要进行计算的地方，必须使用TO_REAL函数将INT类型转换为REAL类型
9. **Timer处理**：所有Timer的PT参数赋值必须使用TO_REAL转换
10. **压力/流量输出**：所有PressureOut和FlowOut的赋值必须使用INT类型（如0而不是0.0）
11. **错误处理**：完善的错误检测和报警机制对于保障设备安全运行至关重要
