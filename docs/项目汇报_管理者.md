# 宝捷信立式注塑机功能库开发项目汇报

---

## 一、项目概况

| 项目 | 内容 |
|------|------|
| **项目名称** | 立式注塑机控制功能库开发 |
| **技术平台** | Luban平台（国产PLC软件，基于Beremiz二次开发） |
| **编程语言** | IEC 61131-3 ST语言 |
| **启动时间** | 2026年4月 |
| **项目目标** | 建立完整的注塑机控制功能库，实现开合模、射出、储料等核心工艺的标准化控制 |

### 项目代码结构

```
Code/
├─ HMI系统                      # HMI接口
├─ 主程序入口                   # PRG_MainControl（唯一主入口）
├─ 全局变量                     # GlobalVars.st
├─ 实例管理                     # Instances.st（顶层实例）
├─ 模式管理                     # 手动/调模/自动 三种模式
├─ 伺服系统                     # 液压轴 FB_HydAxis（8个轴实例）
├─ 合模系统                     # 锁模 + 调模
├─ 注射系统                     # 座台 + 射胶 + 储料
├─ 顶出系统                     # 托模 + 中子
├─ 温度系统                     # EK312温控
└─ 文档                         # 开发规范、架构说明、修改记录
```
