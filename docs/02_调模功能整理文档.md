# 注塑机调模功能整理文档

## 1. 功能概述

调模功能是注塑机的重要功能之一，用于调整模具的闭合距离和锁模力，以适应不同规格的模具和产品生产需求。通过精确的调模控制，可以确保模具闭合紧密，避免产品出现飞边、毛刺等缺陷，同时防止模具和设备受到损坏。

## 2. 功能组成

### 2.1 调模模式
- **手动调模**：通过操作面板手动控制调模电机的前进和后退
- **自动调模**：设定参数后自动完成调模过程
- **电子尺调模**：利用电子尺进行精确定位调模

### 2.2 调模方向
- **调模进**：减小模具闭合距离，增加锁模力
- **调模退**：增大模具闭合距离，减小锁模力

### 2.3 调模控制
- **调模速度控制**：调整调模电机的运行速度
- **调模压力控制**：控制调模过程中的系统压力
- **调模位置控制**：精确控制调模的位置

## 3. 参数说明

### 3.1 调模位置参数
- **调模位置设定**：目标调模位置
- **调模前限**：调模前进的最大位置限制
- **调模后限**：调模后退的最大位置限制
- **调模原点位置**：调模的基准位置

### 3.2 调模速度参数
- **调模快速速度**：调模的快速运行速度
- **调模慢速速度**：调模的慢速运行速度
- **调模精确定位速度**：最终定位时的速度

### 3.3 调模压力参数
- **调模压力设定**：调模过程中的系统压力设定
- **调模低压压力**：低压调模时的压力设定

### 3.4 调模时间参数
- **调模限时**：单次调模操作的最大允许时间
- **调模定位时间**：调模完成后的定位保持时间

## 4. 控制流程

1. **调模准备**：确认模具安装正确，系统压力正常
2. **选择调模模式**：手动/自动/电子尺调模
3. **设定参数**：根据模具规格设定相关参数
4. **执行调模**：按照设定方向进行调模
5. **检测到位**：检测调模是否达到目标位置
6. **压力确认**：确认锁模力是否满足要求
7. **完成调模**：调模完成后锁定参数

## 5. 参数调整原则

### 5.1 位置参数调整
- 调模位置应根据模具厚度进行设置
- 新模具首次使用时应预留足够的安全距离
- 调模前限和后限应根据设备机械结构进行合理设置

### 5.2 速度和压力参数调整
- 调模速度应根据模具大小和重量适当调整
- 大型模具应使用较慢的调模速度
- 调模压力应足够但不宜过高，避免设备过载

### 5.3 调模精度控制
- 最终定位应使用慢速，确保定位精度
- 重要产品生产前应对调模精度进行检查和确认

## 6. 功能实现

### 6.1 控制逻辑
- 调模电机的正反转控制
- 位置检测和反馈
- 速度切换控制
- 压力监控和保护

### 6.2 实现建议
- 采用闭环控制提高调模精度
- 增加调模到位的声音或灯光提示
- 实现调模参数的记忆功能
- 添加调模过程的实时监控画面

## 7. 注意事项

1. **调模前检查**：确保模具和设备处于安全状态
2. **避免过载**：调模压力不应超过设备额定值
3. **定期维护**：定期检查调模机构的机械部件和传感器
4. **参数保护**：关键参数应设置合理的上下限
5. **紧急停止**：发生异常时应立即停止调模操作
6. **记录保存**：重要模具的调模参数应记录保存
