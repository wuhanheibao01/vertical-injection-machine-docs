# 注塑机计时/计数功能整理文档

## 1. 功能概述

计时/计数功能是注塑机的基础辅助功能，用于监控和记录注塑生产过程中的时间和数量参数。通过精确的计时和计数控制，可以提高生产管理效率，优化工艺参数，确保产品质量的一致性，并为设备维护提供数据支持。

## 2. 功能组成

### 2.1 计时功能
- **循环计时**：记录完整注塑循环的时间
- **阶段计时**：记录各工序阶段的时间（如合模、射出、冷却等）
- **延时控制**：设置各工序之间的延时时间
- **定时控制**：设置特定功能的工作时间
- **计时报警**：超出设定时间范围时发出报警

### 2.2 计数功能
- **产品计数**：记录生产的产品数量
- **循环计数**：记录注塑循环次数
- **模具计数**：记录模具使用次数
- **故障计数**：记录各类故障发生次数
- **计数报警**：达到设定计数时发出报警
- **计数维护提醒**：根据计数提醒设备维护

### 2.3 数据管理功能
- **数据记录**：记录生产过程中的时间和计数数据
- **数据统计**：统计和分析生产数据
- **数据清零**：清零各项计时和计数数据
- **数据保持**：断电保持关键数据

## 3. 参数说明

### 3.1 触摸屏参数设置界面
在触摸屏上按【时间 TIME】键，将进入计时/计数设定页面。界面显示以下主要参数：
- 周期时间（设定范围：0.0~6553.5）
- 循环等待时间
- 润滑模数
- 手动动作限时
- 润滑计时
- 故障报警时间
- 系统开机时间（时/分）
- 自动运行时间（时/分）
- 马达运转时间（时/分）

### 3.2 计时参数
- **润滑计时**：表示打油泵单次运行时间
- **循环等待时间**：在全自动过程中，一个动作循环周期结束到下一个动作循环周期开始之间的时间
- **手动动作限时**：手动动作输出时的允许最长时间
- **周期时间**：自动过程中运行周期的限时时间，如果一个循环实际运行的时间超过周期限时，系统报警[周期时间已到]（设定范围：0.0~6553.5）
- **故障报警时间**：发生故障输出时的最长时间，时间到停止报警
- **系统开机时间**：记录系统累计开机运行时间
- **自动运行时间**：记录系统累计自动运行时间
- **马达运转时间**：记录马达累计运转时间
- **冷却时间**：制品冷却时间设定
- **注射过程计时**：记录注射过程的实际时间
- **保压过程计时**：记录保压过程的实际时间

### 3.3 计数参数
- **润滑模数**：计算开模次数，当开模次数到达设定值时，打油泵开始工作
- **产品计数设定**：目标生产数量（默认值：1000次）
- **产品计数当前值**：记录当前实际生产的产品数量
- **不良品计数**：记录生产过程中的不良品数量
- **报警次数计数**：记录设备发生报警的次数
- **计数报警设定**：触发报警的计数设定
- **维护提醒计数**：设备维护提醒的计数设定
- **模具寿命计数**：模具使用次数限制

### 3.4 数据管理参数
- **数据记录间隔**：数据自动记录的时间间隔
- **数据保存数量**：可保存的数据记录数量
- **数据清零权限**：控制数据清零的操作权限

## 4. 控制流程

### 4.1 计时控制流程
1. **开始计时**：循环开始时启动计时
2. **阶段计时**：记录各工序阶段的实际时间
3. **时间监控**：实时监控各阶段时间是否在允许范围内
4. **时间报警**：超出时间限制时发出报警
5. **循环结束**：记录完整循环时间，准备下一个循环

### 4.2 计数控制流程
1. **计数准备**：确认产品完成条件
2. **计数累加**：符合计数条件时累加计数
3. **计数监控**：实时监控各项计数是否达到设定值
4. **计数报警**：达到设定计数时发出报警
5. **计数维护**：根据计数提醒进行设备维护

## 5. 参数调整原则

### 5.1 计时参数调整
- **润滑计时**：应根据设备润滑需求设定，通常为10-30秒，确保充分润滑
- **循环等待时间**：根据生产节奏和辅助操作需要设定，一般为0-10秒
- **手动动作限时**：应设置为正常操作时间的2-3倍，避免误操作导致长时间动作
- **周期时间**：应略长于正常生产周期，建议比实际周期长20-30%，以防止正常波动引起误报警
- **故障报警时间**：根据故障处理流程设定，一般为15-60秒

### 5.2 计数参数调整
- **润滑模数**：根据设备使用频率和润滑要求设定，通常为100-500次开模动作
- **产品计数设定**：根据生产计划和订单要求设定
- **维护提醒计数**：根据设备制造商的建议设定，通常为定期维护的间隔周期

## 7. 注意事项

1. **数据准确性**：确保计时和计数数据的准确性和可靠性，特别是周期时间设置应合理
2. **数据保护**：重要数据（如累计运行时间）应保存在掉电保持区域，确保断电后数据不丢失
3. **权限管理**：计数清零等关键操作应设置权限控制，防止误操作
4. **定期校准**：计时功能应定期校准以确保精度，特别是对生产周期要求严格的场合
5. **故障处理**：出现计时/计数异常时应及时检查和处理，特别是周期时间报警
6. **数据备份**：重要生产数据应定期备份，以便后续分析和追溯
7. **润滑管理**：根据润滑模数和润滑计时参数确保设备得到定期润滑，延长设备寿命
8. **参数范围**：周期时间设定范围为0.0~6553.5，超出范围将无法正常设置
9. **手动操作安全**：手动动作限时参数设置应合理，防止手动操作时发生意外
10. **报警及时处理**：故障报警时间到后会自动停止报警，但仍需及时处理故障原因
