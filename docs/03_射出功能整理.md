# 注塑机射出功能

## 1. 概述

### 1.1 功能简介

<div class="info-card">
  <div class="icon">⚡</div>
  <strong>核心功能</strong>
  <p>射出功能是注塑机的核心功能之一，负责将熔融状态的塑料通过注射螺杆高压高速地注入模具型腔。射出过程的控制精度直接影响产品的质量、尺寸稳定性和生产效率。</p>
</div>

### 1.2 工艺特点

<div class="feature-block">
  <h4>工艺特性</h4>
  <ul>
    <li><strong>多段射出控制</strong>：支持10段射出工艺参数，可实现复杂的填充曲线</li>
    <li><strong>多段保压控制</strong>：支持8段保压工艺参数，确保制品尺寸稳定性</li>
    <li><strong>射退功能</strong>：支持射退动作，防止浇口拉丝和制品缺陷</li>
    <li><strong>多种控制方式</strong>：支持电子尺模式、行程模式、时间模式三种控制方式</li>
    <li><strong>斜率控制</strong>：支持压力和速度的启动/停止斜率控制</li>
    <li><strong>安全机制</strong>：包含超时保护、状态互锁等多重安全保障</li>
    <li><strong>平台兼容性</strong>：支持Luban平台（基于Beremiz二次开发）运行，采用标准IEC 61131-3 ST语法实现</li>
  </ul>
</div>

### 1.3 技术架构

## 2. 核心控制机制

### 2.1 射出到位判断机制

<div class="info-card">
  <div class="icon">✅</div>
  <strong>多模式到位判断</strong>
  <p>射出到位判断支持三种模式：电子尺模式、行程模式和时间模式，根据实际工艺需求选择合适的控制方式。</p>
</div>

```mermaid
flowchart TD
    Start[射出阶段开始] --> ModeSelect{射出方式选择}

    ModeSelect -->|电子尺模式| PosCheck{位置判断}
    ModeSelect -->|行程模式| DICheck{行程开关判断}
    ModeSelect -->|时间模式| TimeCheck{时间判断}

    PosCheck -->|电子尺位置到达| InjComplete[射出完成]
    DICheck -->|行程开关触发| InjComplete
    TimeCheck -->|段时间到| InjComplete

    class PosCheck,DICheck,TimeCheck decision
    class InjComplete complete
```

1. **电子尺位置判断**：通过电子尺反馈的螺杆位置值判断
   - 触发条件：`电子尺位置值 >= 设定位置`
   - 对应参数：aInjSeg[1..10].udiPos

2. **行程开关判断**：通过外部DI传感器信号直接检测
   - 触发条件：外部行程开关信号触发
   - 对应参数：bInjSeg1Stop、bInjSeg2Stop、bInjStop

3. **时间判断**：通过计算各阶段的持续时间
   - 触发条件：各段时间达到设定值
   - 对应参数：aInjSeg[1..10].uiTime

### 2.2 电子尺功能说明

<div class="feature-block">
  <h4>电子尺作用</h4>
  <ul>
    <li><strong>位置反馈</strong>：通过电子尺实时反馈注射螺杆的实际位置</li>
    <li><strong>阶段切换控制</strong>：通过电子尺反馈的位置值控制各阶段的切换</li>
    <li><strong>参数映射</strong>：射出目标位置、各段位置等参数用于阶段控制</li>
  </ul>
</div>

## 4. 控制流程

### 4.1 射出过程流程

#### 4.1.1 射出流程示意图

```mermaid
flowchart TD
    Start[初始状态] -->|启动信号| Init[初始化]
    Init -->|初始化完成| Idle[空闲状态]
    Idle -->|射出模式启动| InjectMode[射出等级选择 1-N级]

    subgraph 射出 N级 循环
        direction TB
        InjectMode --> InitInj[初始化当前等级=1]
        InitInj --> ExecuteInj[执行射出等级 当前等级]
        ExecuteInj -->|位置到达或时间到| CheckInj{当前等级 ≤ 选择等级?}
        CheckInj -->|否| EndInjLoop[结束射出循环]
        CheckInj -->|是| IncrementInj[当前等级+1]
        IncrementInj --> ExecuteInj
    end

    EndInjLoop -->|射出完成| HoldMode[保压等级选择 1-M级]

    subgraph 保压 N级 循环
        direction TB
        HoldMode --> InitHold[初始化当前等级=1]
        InitHold --> ExecuteHold[执行保压等级 当前等级]
        ExecuteHold -->|时间到| CheckHold{当前等级 ≤ 选择等级?}
        CheckHold -->|否| EndHoldLoop[结束保压循环]
        CheckHold -->|是| IncrementHold[当前等级+1]
        IncrementHold --> ExecuteHold
    end

    EndHoldLoop -->|保压完成| End[射出流程结束]

    InjectMode -->|错误| Error[错误状态]
    ExecuteInj -->|超时| Error
    ExecuteHold -->|超时| Error

    Error -->|复位| Idle

    class InitInj,ExecuteInj,IncrementInj,EndInjLoop step
    class InitHold,ExecuteHold,IncrementHold,EndHoldLoop step
    class InjectMode,HoldMode,Start,Init,Idle,End complete
    class CheckInj,CheckHold decision
    class Error error
```

#### 4.1.2 射出流程序列图

```mermaid
sequenceDiagram
    participant 系统
    participant 射出控制
    participant 执行机构

    系统->>射出控制: 启动信号 (bStart=TRUE)
    射出控制->>射出控制: 初始化 (eState_Init)
    射出控制->>射出控制: 射出等级选择 (1-N级)
    射出控制->>射出控制: 初始化当前等级=1

    loop 射出等级循环
        射出控制->>执行机构: 执行射出等级(当前等级)
        alt 位置到达或时间到
            执行机构-->>射出控制: 位置到达或时间到信号
        else 超时
            执行机构-->>射出控制: 超时信号
            射出控制->>射出控制: 错误处理 (eState_Error)
        end
        射出控制->>射出控制: 当前等级+1
        射出控制->>射出控制: 检查是否达到选择等级
    end

    射出控制->>射出控制: 保压等级选择 (1-M级)
    射出控制->>射出控制: 初始化当前等级=1

    loop 保压等级循环
        射出控制->>执行机构: 执行保压等级(当前等级)
        alt 时间到
            执行机构-->>射出控制: 保压段时间到信号
        else 超时
            执行机构-->>射出控制: 超时信号
            射出控制->>射出控制: 错误处理 (eState_Error)
        end
        射出控制->>射出控制: 当前等级+1
        射出控制->>射出控制: 检查是否达到选择等级
    end

    射出控制-->>系统: 完成信号 (bDone=TRUE, bInjected=TRUE, bHeld=TRUE)
```

### 4.2 射退过程流程

#### 4.2.1 射退流程示意图

```mermaid
flowchart TD
    Start[初始状态] -->|启动信号| Init[初始化]
    Init -->|初始化完成| Idle[空闲状态]
    Idle -->|射退模式启动| SuckBackMode[射退准备]
    SuckBackMode --> Execute[执行射退动作]
    Execute -->|位置到达| CheckPos{位置到达?}
    CheckPos -->|否| CheckTime{时间到?}
    CheckPos -->|是| SuckBacked[射退完成]
    CheckTime -->|否| CheckTime
    CheckTime -->|是| SuckBacked
    Execute -->|超时| Error[错误状态]
    Error -->|复位| Idle
    SuckBacked -->|复位| Idle

    class Start,Init,Idle,SuckBackMode,Execute,SuckBacked state
    class CheckPos,CheckTime decision
    class Error error
```

#### 4.2.2 射退流程序列图

```mermaid
sequenceDiagram
    participant 系统
    participant 射出控制
    participant 执行机构

    系统->>射出控制: 射退启动信号 (bStart=TRUE, uiInjectMode=2)
    射出控制->>射出控制: 初始化 (eState_Init)
    射出控制->>执行机构: 射退动作 (eState_SuckBacking)

    alt 位置到达
        执行机构-->>射出控制: 射退停止信号 (bSuckBackStop=TRUE)
        射出控制->>射出控制: 射退完成 (eState_SuckBacked)
        射出控制-->>系统: 完成信号 (bDone=TRUE, bSuckBacked=TRUE)
    else 时间到
        执行机构-->>射出控制: 射退时间到信号
        射出控制->>射出控制: 射退完成 (eState_SuckBacked)
        射出控制-->>系统: 完成信号 (bDone=TRUE, bSuckBacked=TRUE)
    else 开关触发
        执行机构-->>射出控制: 射退到位开关信号
        射出控制->>射出控制: 射退完成 (eState_SuckBacked)
        射出控制-->>系统: 完成信号 (bDone=TRUE, bSuckBacked=TRUE)
    else 超时
        射出控制->>射出控制: 错误处理 (eState_Error)
        射出控制-->>系统: 错误信号 (bAlarm=TRUE, uiAlarmID=1003)
    end
```

> ⚠️ **重要说明**：
>
> 1. 射出等级段数可通过`uiInjSegCnt`参数设定（1-10段）
> 2. 保压等级段数可通过`uiHoldSegCnt`参数设定（1-8段）
> 3. 射退可在保压完成后执行，防止浇口拉丝

#### 转保压方式说明

转保压触发采用**按位组合选择**模式，允许用户同时选择1-3种触发方式，满足任一条件即触发转保压。

| 位 | 值 | 触发方式 | 说明 |
|----|-----|---------|------|
| Bit0 | 1 | 时间触发 | 达到设定的 `uiInjToHoldTime` 时间后触发 |
| Bit1 | 2 | 压力触发 | 达到设定的 `uiInjToHoldPres` 压力后触发 |
| Bit2 | 4 | 位置触发 | 达到设定的 `udiInjToHoldPos` 位置后触发 |

**组合示例**：

| dwInjToHoldMode | 触发条件 | 说明 |
|-----------------|---------|------|
| 1 | 仅时间 | 只用时间判断 |
| 2 | 仅压力 | 只用压力判断 |
| 4 | 仅位置 | 只用位置判断 |
| 3 (1+2) | 时间或压力 | 两者任一满足即触发 |
| 5 (1+4) | 时间或位置 | 两者任一满足即触发 |
| 6 (2+4) | 压力或位置 | 两者任一满足即触发 |
| 7 (1+2+4) | 时间或压力或位置 | 三者任一满足即触发 |

**使用示例**：
- `dwInjToHoldMode = 3` 表示同时启用时间和压力判断，射出过程中达到任一阈值即转入保压阶段

## 8. 安全保护机制

### 8.1 超时保护

- **射出总时间保护**：监控整个射出和保压过程，超过设定总时间则报警
- **各段超时保护**：各段工艺参数都应有合理的时间限制

### 8.2 位置保护

- **电子尺位置监控**：实时监控注射螺杆位置
- **位置极限保护**：防止超出机械行程极限

### 8.3 状态互锁

- **状态机互锁**：各状态间应有明确的转换条件，防止异常跳转
- **命令互锁**：启动、停止、急停命令应有优先级处理

## 10. 参数调整指南

### 10.1 射出参数调整

<div class="info-card">
  <div class="icon">💡</div>
  <strong>调整建议</strong>
  <p>射出参数的调整应根据产品材质、模具结构和制品要求进行优化。</p>
</div>

1. **压力调整**：根据制品重量和材质选择合适的射出压力
   - 首段压力不宜过高，防止溅料
   - 后段压力适当增加，确保填充完整

2. **速度调整**：根据填充阶段调整速度
   - 首段速度较慢，保证熔融塑料平稳进入模具
   - 中段速度加快，提高生产效率
   - 末段速度减慢，确保模具填充完整

3. **位置调整**：电子尺位置用于精确控制各段切换点
   - 确保各段位置设置合理，避免欠注或过填充

### 10.2 保压参数调整

1. **保压压力**：根据制品收缩特性调整
   - 收缩率大的材质需要较高保压压力
   - 保压压力应逐段递减

2. **保压时间**：根据制品厚度和材质调整
   - 厚制品需要较长的保压时间
   - 保压时间不足会导致制品缩水

### 10.3 射退参数调整

1. **射退位置**：防止浇口拉丝
   - 射退距离不宜过大，防止空气进入
   - 射退速度应适中

## 11. 调试与故障排除

### 11.1 常见问题与解决方案

| 问题现象 | 可能原因 | 解决建议 |
|----------|----------|----------|
| 射出无力 | 压力参数设置过低 | 检查并调整射出压力参数 |
| 制品欠注 | 射出速度太慢或位置设置不当 | 调整射出速度和各段位置参数 |
| 制品过填充 | 射出压力过高或时间过长 | 减少射出压力或缩短射出时间 |
| 保压不足导致缩水 | 保压压力或时间不够 | 增加保压压力或延长保压时间 |
| 射退拉丝 | 射退距离或速度不当 | 调整射退参数 |
| 射出超时报警 | 各段时间设置过长 | 优化各段时间参数 |

### 11.2 报警处理

| 报警代码 | 说明 | 处理方法 |
|----------|------|----------|
| 1001 | 射出超时 | 检查射出各段时间设置，检查模具和螺杆状态 |
| 1002 | 保压超时 | 检查保压各段时间设置 |
| 1003 | 射退超时 | 检查射退位置和时间参数 |

## 12. 数据流说明

```mermaid
graph LR
    subgraph 输入层
        I1[工艺参数设定]
        I2[控制命令]
        I3[传感器反馈]
    end

    subgraph 处理层
        P1[参数校验]
        P2[状态机控制]
        P3[逻辑运算]
    end

    subgraph 输出层
        O1[压力命令]
        O2[速度命令]
        O3[位置命令]
    end

    I1 --> P1
    I2 --> P2
    I3 --> P3
    P1 --> P2
    P2 --> P3
    P3 --> O1
    P3 --> O2
    P3 --> O3
```

1. **输入层**：接收工艺参数设定、控制命令和传感器反馈
2. **处理层**：进行参数校验、状态机控制和逻辑运算
3. **输出层**：生成压力、速度和位置命令到液压驱动
