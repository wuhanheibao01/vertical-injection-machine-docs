## 1. 功能概述

### 1.1 功能简介

<div class="info-card">
  <div class="icon">⚡</div>
  <strong>核心功能</strong>
  <p>射出功能是注塑机的核心功能之一，负责将熔融状态的塑料通过注射螺杆高压高速地注入模具型腔。射出过程的控制精度直接影响产品的质量、尺寸稳定性和生产效率。本功能支持10段射出、8段保压以及射退控制，通过精确控制射出压力、速度、位置和时间等参数，实现高质量的塑料注射成型。</p>
</div>

### 1.2 工艺特点

- **多段射出控制**：支持10段射出工艺参数，可实现复杂的填充曲线
- **多段保压控制**：支持8段保压工艺参数，确保制品尺寸稳定性
- **射退功能**：支持射退动作，防止浇口拉丝和制品缺陷
- **多种控制方式**：支持电子尺模式、行程模式、时间模式三种控制方式
- **斜率控制**：支持压力和速度的启动/停止斜率控制
- **安全机制**：包含超时保护、状态互锁等多重安全保障
- **平台兼容性**：支持Luban平台（基于Beremiz二次开发）运行，采用标准IEC 61131-3 ST语法实现

## 2. 核心控制机制

### 2.1 状态管理机制

射出功能采用统一状态机管理，包含射出、保压和射退三个主要阶段：

#### 2.1.1 射出状态机 (E_InjectState)

```mermaid
flowchart TD
    Start[初始状态 eState_Idle] -->|启动信号| Init[初始化状态 eState_Init]
    Init -->|初始化完成| Idle[空闲状态 eState_Idle]
    Idle -->|射出模式启动| Injecting[射出中 eState_Injecting]
    Injecting -->|射出10段完成| Injected[射出完成 eState_Injected]
    Injected -->|转保压条件满足| Holding[保压中 eState_Holding]
    Holding -->|保压8段完成| Held[保压完成 eState_Held]
    Held -->|射出退模式启动| SuckBacking[射退中 eState_SuckBacking]
    SuckBacking -->|射退完成| SuckBacked[射退完成 eState_SuckBacked]
    SuckBacked -->|复位| Idle
    Injecting -->|错误发生| Error[错误状态 eState_Error]
    Holding -->|错误发生| Error
    SuckBacking -->|错误发生| Error
    Error -->|复位命令| Idle

    class Start,Init,Idle,Injecting,Injected,Holding,Held,SuckBacking,SuckBacked state
    class Error error
```

#### 2.1.2 状态说明

| 状态值 | 名称 | 说明 |
|--------|------|------|
| 0 | eState_Idle | 空闲状态 |
| 1 | eState_Init | 初始化状态 |
| 2 | eState_Injecting | 射出中(10段) |
| 3 | eState_Injected | 射出完成 |
| 4 | eState_Holding | 保压中(8段) |
| 5 | eState_Held | 保压完成 |
| 6 | eState_SuckBacking | 射退中 |
| 7 | eState_SuckBacked | 射退完成 |
| 8 | eState_Error | 错误状态 |

### 2.2 控制命令机制

| 命令 | 说明 | 响应 |
|------|------|------|
| bStart | 启动命令 | 启动射出/保压/射退动作 |
| bStop | 停止命令 | 有减速停止 |
| bEStop | 急停命令 | 立即停止，无减速 |
| bReset | 复位命令 | 重置错误状态 |

### 2.3 模式选择机制

| 模式值 | 名称 | 说明 |
|--------|------|------|
| 0 | 无模式 | 不执行任何动作 |
| 1 | 射出模式 | 执行射出+保压动作 |
| 2 | 射退模式 | 执行射退动作 |

### 2.4 射出方式选择 (uiInjMode)

| 方式值 | 名称 | 说明 |
|--------|------|------|
| 0 | 电子尺模式 | 通过电子尺位置反馈控制 |
| 1 | 行程模式 | 通过行程开关控制 |
| 2 | 时间模式 | 通过时间控制各段切换 |

## 4. 控制流程

### 4.1 射出过程流程

#### 4.1.1 射出流程示意图

```mermaid
flowchart TD
    Start[初始状态] -->|启动信号| Init[初始化]
    Init -->|初始化完成| Idle[空闲状态]
    Idle -->|射出模式启动| InjectMode[射出等级选择 1-N级]

    subgraph 射出 N级 循环
        direction TB
        InjectMode --> InitInj[初始化当前等级=1]
        InitInj --> ExecuteInj[执行射出等级 当前等级]
        ExecuteInj -->|位置到达或时间到| CheckInj{当前等级 ≤ 选择等级?}
        CheckInj -->|否| EndInjLoop[结束射出循环]
        CheckInj -->|是| IncrementInj[当前等级+1]
        IncrementInj --> ExecuteInj
    end

    EndInjLoop -->|射出完成| HoldMode[保压等级选择 1-M级]

    subgraph 保压 N级 循环
        direction TB
        HoldMode --> InitHold[初始化当前等级=1]
        InitHold --> ExecuteHold[执行保压等级 当前等级]
        ExecuteHold -->|时间到| CheckHold{当前等级 ≤ 选择等级?}
        CheckHold -->|否| EndHoldLoop[结束保压循环]
        CheckHold -->|是| IncrementHold[当前等级+1]
        IncrementHold --> ExecuteHold
    end

    EndHoldLoop -->|保压完成| End[射出流程结束]

    InjectMode -->|错误| Error[错误状态]
    ExecuteInj -->|超时| Error
    ExecuteHold -->|超时| Error

    Error -->|复位| Idle

    class InitInj,ExecuteInj,IncrementInj,EndInjLoop step
    class InitHold,ExecuteHold,IncrementHold,EndHoldLoop step
    class InjectMode,HoldMode,Start,Init,Idle,End complete
    class CheckInj,CheckHold decision
    class Error error
```

#### 4.1.2 射出流程序列图

```mermaid
sequenceDiagram
    participant 系统
    participant 射出控制
    participant 执行机构

    系统->>射出控制: 启动信号 (bStart=TRUE)
    射出控制->>射出控制: 初始化 (eState_Init)
    射出控制->>射出控制: 射出等级选择 (1-N级)
    射出控制->>射出控制: 初始化当前等级=1

    loop 射出等级循环
        射出控制->>执行机构: 执行射出等级(当前等级)
        alt 位置到达或时间到
            执行机构-->>射出控制: 位置到达或时间到信号
        else 超时
            执行机构-->>射出控制: 超时信号
            射出控制->>射出控制: 错误处理 (eState_Error)
        end
        射出控制->>射出控制: 当前等级+1
        射出控制->>射出控制: 检查是否达到选择等级
    end

    射出控制->>射出控制: 保压等级选择 (1-M级)
    射出控制->>射出控制: 初始化当前等级=1

    loop 保压等级循环
        射出控制->>执行机构: 执行保压等级(当前等级)
        alt 时间到
            执行机构-->>射出控制: 保压段时间到信号
        else 超时
            执行机构-->>射出控制: 超时信号
            射出控制->>射出控制: 错误处理 (eState_Error)
        end
        射出控制->>射出控制: 当前等级+1
        射出控制->>射出控制: 检查是否达到选择等级
    end

    射出控制-->>系统: 完成信号 (bDone=TRUE, bInjected=TRUE, bHeld=TRUE)
```

### 4.2 射退过程流程

#### 4.2.1 射退流程示意图

```mermaid
flowchart TD
    Start[初始状态] -->|启动信号| Init[初始化]
    Init -->|初始化完成| Idle[空闲状态]
    Idle -->|射退模式启动| SuckBackMode[射退准备]
    SuckBackMode --> Execute[执行射退动作]
    Execute -->|位置到达| CheckPos{位置到达?}
    CheckPos -->|否| CheckTime{时间到?}
    CheckPos -->|是| SuckBacked[射退完成]
    CheckTime -->|否| CheckTime
    CheckTime -->|是| SuckBacked
    Execute -->|超时| Error[错误状态]
    Error -->|复位| Idle
    SuckBacked -->|复位| Idle

    class Start,Init,Idle,SuckBackMode,Execute,SuckBacked state
    class CheckPos,CheckTime decision
    class Error error
```

#### 4.2.2 射退流程序列图

```mermaid
sequenceDiagram
    participant 系统
    participant 射出控制
    participant 执行机构

    系统->>射出控制: 射退启动信号 (bStart=TRUE, uiInjectMode=2)
    射出控制->>射出控制: 初始化 (eState_Init)
    射出控制->>执行机构: 射退动作 (eState_SuckBacking)

    alt 位置到达
        执行机构-->>射出控制: 射退停止信号 (bSuckBackStop=TRUE)
        射出控制->>射出控制: 射退完成 (eState_SuckBacked)
        射出控制-->>系统: 完成信号 (bDone=TRUE, bSuckBacked=TRUE)
    else 时间到
        执行机构-->>射出控制: 射退时间到信号
        射出控制->>射出控制: 射退完成 (eState_SuckBacked)
        射出控制-->>系统: 完成信号 (bDone=TRUE, bSuckBacked=TRUE)
    else 开关触发
        执行机构-->>射出控制: 射退到位开关信号
        射出控制->>射出控制: 射退完成 (eState_SuckBacked)
        射出控制-->>系统: 完成信号 (bDone=TRUE, bSuckBacked=TRUE)
    else 超时
        射出控制->>射出控制: 错误处理 (eState_Error)
        射出控制-->>系统: 错误信号 (bAlarm=TRUE, uiAlarmID=1003)
    end
```

> ⚠️ **重要说明**：
>
> 1. 射出等级段数可通过`uiInjSegCnt`参数设定（1-10段）
> 2. 保压等级段数可通过`uiHoldSegCnt`参数设定（1-8段）
> 3. 射退可在保压完成后执行，防止浇口拉丝

## 8. 安全保护机制

### 8.1 超时保护

- **射出总时间保护**：监控整个射出和保压过程，超过设定总时间则报警
- **各段超时保护**：各段工艺参数都应有合理的时间限制

### 8.2 位置保护

- **电子尺位置监控**：实时监控注射螺杆位置
- **位置极限保护**：防止超出机械行程极限

### 8.3 状态互锁

- **状态机互锁**：各状态间应有明确的转换条件，防止异常跳转
- **命令互锁**：启动、停止、急停命令应有优先级处理

## 9. 参数调整指南

### 9.1 射出参数调整

1. **段数选择**：根据制品复杂程度选择合适的射出段数
2. **压力调整**：根据材料流动性调整，压力应确保完全填充
3. **速度调整**：速度影响填充时间和制品表面质量
4. **位置调整**：位置切换点应根据制品形状和壁厚调整
5. **斜率调整**：斜率控制可使动作更平滑，减少冲击

### 9.2 保压参数调整

1. **保压压力**：应略低于射出最终压力，防止制品过压
2. **保压时间**：根据制品壁厚和材料特性调整
3. **段数选择**：复杂制品可增加保压段数，细化压力控制

### 9.3 射退参数调整

1. **射退时机**：射退应在保压完成后执行
2. **射退距离**：根据浇口大小和制品特性调整
3. **射退速度**：应适中，防止产生气泡或制品变形

## 10. 调试与故障排除

### 10.1 常见问题

| 问题现象 | 可能原因 | 解决方法 |
|----------|----------|----------|
| 射出不到位 | 压力不足或速度过快 | 调整压力和速度参数 |
| 制品有气穴 | 射出速度过快或模具排气不良 | 降低射出速度，改善模具排气 |
| 制品缩水 | 保压压力不足或保压时间过短 | 增加保压压力或延长保压时间 |
| 射退后拉丝 | 射退速度过快或温度过高 | 降低射退速度或调整温度 |

### 10.2 调试建议

1. **参数记录**：每次调整参数后应记录，便于问题追溯
2. **分段调试**：先调试单段，确认正常后再启用多段
3. **数据监控**：利用电子尺反馈数据监控射出过程
4. **模具检查**：定期检查模具状况，确保排气良好
