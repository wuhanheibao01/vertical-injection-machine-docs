# 注塑机绞牙功能整理文档

## 1. 功能概述

绞牙功能是注塑机的特殊功能之一，主要用于具有螺纹结构的塑料制品成型。该功能通过控制绞牙电机的旋转运动，实现螺纹型芯的旋转脱模，确保带有螺纹的产品能够顺利从模具中脱出而不损坏产品。绞牙功能通常与中子功能配合使用，是复杂注塑产品成型的重要技术手段。

## 2. 功能组成

### 2.1 绞牙控制模式
- **位置控制模式**：通过编码器精确控制绞牙旋转角度
- **时间控制模式**：通过控制绞牙旋转时间来控制旋转量
- **压力控制模式**：通过控制绞牙压力来防止过载

### 2.2 绞牙动作流程
- **绞牙前进**：螺纹型芯旋转进入模具
- **绞牙后退**：螺纹型芯旋转退出模具
- **绞牙定位**：精确控制绞牙旋转位置
- **绞牙刹车**：快速停止绞牙旋转运动

### 2.3 绞牙辅助功能
- **绞牙监控**：实时监控绞牙过程参数
- **同步控制**：与开模/合模动作的同步控制
- **异常保护**：检测并处理绞牙过程中的异常情况

## 3. 参数说明

### 3.1 绞牙位置参数
- **绞牙前进角度**：绞牙前进的总旋转角度
- **绞牙后退角度**：绞牙后退的总旋转角度
- **绞牙定位角度**：绞牙精确定位的角度
- **绞牙原点位置**：绞牙的基准位置

### 3.2 绞牙速度参数
- **绞牙前进速度**：绞牙前进的旋转速度
- **绞牙后退速度**：绞牙后退的旋转速度
- **绞牙定位速度**：绞牙精确定位时的速度

### 3.3 绞牙压力参数
- **绞牙前进压力**：绞牙前进时的压力限制
- **绞牙后退压力**：绞牙后退时的压力限制
- **绞牙过载保护**：绞牙过载时的保护压力

### 3.4 时间参数
- **绞牙前进时间**：绞牙前进的最大允许时间
- **绞牙后退时间**：绞牙后退的最大允许时间
- **绞牙等待时间**：绞牙动作之间的等待时间

### 3.5 同步参数
- **开模与绞牙同步位置**：开模到什么位置开始绞牙后退
- **合模与绞牙同步位置**：合模到什么位置开始绞牙前进
- **绞牙完成确认时间**：确认绞牙完成的等待时间

## 4. 控制流程

### 4.1 绞牙前进流程
1. **前进准备**：确认模具闭合到位，系统压力正常
2. **绞牙前进**：绞牙电机开始旋转前进
3. **速度控制**：按照设定速度进行旋转
4. **位置检测**：检测绞牙是否达到设定角度
5. **完成前进**：绞牙前进到位后停止并定位

### 4.2 绞牙后退流程
1. **后退准备**：确认开模到达同步位置
2. **绞牙后退**：绞牙电机开始旋转后退
3. **速度控制**：按照设定速度进行旋转
4. **位置检测**：检测绞牙是否达到设定角度
5. **完成后退**：绞牙后退到位后停止并定位

## 5. 参数调整原则

### 5.1 位置参数调整
- 绞牙角度应根据产品螺纹的导程和圈数精确计算
- 新模具首次使用时应预留一定的调整余量
- 绞牙原点位置应定期校准

### 5.2 速度参数调整
- 绞牙速度应根据产品材料特性和螺纹结构调整
- 高粘度材料或复杂螺纹应使用较低的绞牙速度
- 定位阶段应使用较低速度以确保定位精度

### 5.3 压力参数调整
- 绞牙压力应足够但不宜过高，避免电机过载
- 过载保护压力应设置为正常工作压力的1.2-1.5倍
- 不同材料和螺纹结构需要不同的压力设置

### 5.4 同步参数调整
- 开模与绞牙同步位置应确保模具打开足够距离
- 合模与绞牙同步位置应确保模具闭合足够紧密
- 同步时机的调整直接影响产品质量和生产效率

## 6. 功能实现

### 6.1 控制逻辑
- 绞牙电机的正反转控制
- 旋转角度的精确检测和控制
- 与开模/合模动作的同步控制
- 过载保护和异常处理

### 6.2 实现建议
- 使用高精度编码器实现精确角度控制
- 采用闭环控制算法提高绞牙精度
- 实现绞牙过程的实时监控和参数显示
- 添加绞牙异常的自动检测和报警功能

## 7. 注意事项

1. **同步控制**：确保绞牙动作与开模/合模动作的正确同步
2. **过载保护**：设置合理的过载保护参数，防止设备损坏
3. **定期维护**：定期检查绞牙电机和传动机构
4. **参数校准**：定期校准绞牙角度和位置
5. **异常处理**：发生绞牙异常时应立即停止并报警
6. **模具保护**：确保绞牙动作不会损坏模具
